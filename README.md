# Al-Hafiz - Quran Memorization App

## Overview
Al-Hafiz is a premium web application designed to assist in memorizing the Holy Quran. It features a beautiful, ethereal Islamic interface, a structured memorization plan, and tools for recitation and review.

## Features
- **Premium UI/UX**: Glassmorphism design with deep midnight blue and gold accents.
- **Memorization Dashboard**: Track daily progress and streaks.
- **Quran Memorization Tool**: 
  - Indo-Pak script display.
  - Audio recitation by Mishari Rashid Alafasy.
  - "Hide Text" mode for testing memory.
  - "Recite Back" simulation.
- **Structured Plan**: Day-by-day memorization schedule.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Technologies
- React
- Vite
- Lucide React (Icons)
- Framer Motion (Animations)
- Quran.com API (for verses and audio)
